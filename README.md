# 🎮 Tidy Tuesday — Pokémon Stats Analysis (Week 13, 2025)
**Author:** Bukola Elliott | **Date:** April 2025

This is my TidyTuesday submission for the Pokémon dataset (Week 13, 2025),
completed as part of my M.A. Economics with Data Analysis concentration 
at Georgia State University.

## Research Questions
- Which Pokémon types are fastest and strongest on average?
- How does legendary status vary across generations?
- What type-stat synergies dominate competitive play?

## Key Findings
- Flying and Electric types lead in average Speed (Noivern, Electrode)
- Dragon and Fighting types dominate average Attack (Rayquaza-Mega, Lucario-Mega)
- Fairy types rank last in both Speed and Attack
- Generation 4 introduced the most Legendary Pokémon
